# seoultech-computervision-video-recorder

컴퓨터비전 2주차 과제 - Video Recorder

## Description

CCTV의 실시간스트리밍 주소를 통해 영상을 재생 및 녹화하여 파일로 저장 가능한 프로그램

---

### Mods

각 모드 별 기능

- Preview Mod: 미리보기 화면 출력
- Recorder Mod: 모드 전환 시 녹화 시작
  - 출력 화면에 녹화 여부 표시
  - 녹화 시작 시간을 파일명으로 저장: %Y-%m-%d %H:%M:%S

---

### Keys

프로그램에 할당된 키

- ESC: 프로그램 종료
- Space: 모드 전환
